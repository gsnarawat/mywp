##################### Installation #####################
1. Upload the `Innovative` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options' to change theme options



############# User Guide and Main Features #############
=== WP Version ===
As this theme uses the WordPress Innovative feature for most options, it requires at least version 3.4 to work properly.
Please make sure your WP version is at least 3.4.


=== Upload a logo ===
Go to the Theme Option screen in the logo section and upload your image.
For best results, try uploading a logo with the following maximum dimensions => max-height :100px, max-width ; 300px

=== Select a menu ===
Innovative comes with four menu locations. Select a menu in the appearance=> menus section

=== Social links ===
You can set up to nine social network profile in the Innovative option screen. Write the url of your social profile in the fields.
